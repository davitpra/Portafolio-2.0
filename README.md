## Running Locally

```bash
npm install
```

Create `.env.local`:
```
NEXT_PUBLIC_SITE_URL=https://davitprado.com
```

```bash
npm run dev   # http://localhost:3000
npm run build
npm run lint
```

Built with [Next.js](https://nextjs.org) and [Tailwind CSS](https://tailwindcss.com).
